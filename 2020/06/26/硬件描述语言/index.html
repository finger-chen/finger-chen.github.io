整数型常量

  • 十进制数形式
  • 带基数形式<+/->‘
  • 最左边是最高有效位，最右边是最低有效位*

eg:

3’b101
4’shf
5’o37
8’he3
8’b1001_0011
12’h13x

实数型常量

  • 十进制计数法
  • 科学记数法

eg:

5E-4 == 0.0005
23_5.1e2 == 23510.0
e与E相同

字符串常量
用双撇号括起来，必须==在同一行==中不能分成多行书写。

一个字符对应八位的ASCII值，是无符号整数

转义符

\n
\t
\
\“

符号常量

用参数定义语句定义的常量
parameter 参数名1=常量表达式1，参数名2=常量表达式2······

eg:parameter BIT=1, BYTE=8, PI=3.14

BIT&BYTE的位宽为0~31
parameter <name> = <value>;
位于module内部，常用于指定延迟，变量的位宽，状态机的状态值.

位于module外部，是全局常量，作用范围从定义开始到整个程序结束。

数据类型

变量的数据类型：

线网类型
寄存器类型

线网类型变量
表示硬件电路中元件之间实际存在的物理连线

  • 常用的类型是==wire==
  • Signed缺省时，为无符号数
  • 位宽[msb:lsb]缺省时，为1位

线网类型 功能说明
wire, tri 表示元件之间的连线，wire为一般连线；tri描述由多个型号源驱动的线网
wor, trior 具有线或特性的线网
wand, triand 具有线与特性的线网
trireg 具有电荷保持特性的线网类型，用于开关级建模
tri1 上拉电阻，用于开关级建模
tri0 下拉电阻，用于开关级建模
supply1 用于对电源建模，高电平1
supply0 用于对地建模，低电平0
  • 线网类型变量的值由驱动元件差不多就是生成这个变量的门的值决定
  • 如果没有驱动元件连接到线网，则线网的缺省值为z
  • 线网trireg,它的默认值为x
  • 避免多重驱动，因为这样逻辑值不能确定*

寄存器类型变量

  • 表示一个抽象的数据存储单元
  • 只能在initial或always内部中被赋值
  • 值从一条赋值语句保持到下一条赋值语句
寄存器类型 含义
reg 寄存器型变量，默认值为x
integer 32位带符号的整数型变量，默认值为x
real/realtime 64位带符号的实数型变量，默认值为0
time 64位无符号的时间变量，默认值为x

寄存器变量占用物理内存
线网型变量不占用物理内存，有其他变量决定

条件语句（if语句）

有三种形式的if语句

  • if后面的条件表达式一般为逻辑表达式或者关系表达式
  • 表达式的值如果为==1则按真处理==
  • 如果表达式的值为0、X、Z则按假处理

使用if - else语句对4选1数据选择器的行为进行描述

因为过程赋值语句只能给寄存器型变量赋值，所以输出变量Y的数据类型定义为reg

一般形式

关键词casex和czsez表示含有无关项X和高组态Z的情况
